📅 Firebase Setup

One-time setup to enable shared data for your team. Takes about 15 minutes.

1

Create a Firebase project

Go to console.firebase.google.com → "Add project" → Enter any name (e.g. lab-meeting) → Disable Google Analytics → Create

2

Enable Realtime Database

Left menu → "Build" → "Realtime Database" → "Create database" → Choose a region → Select "Start in test mode" → Enable

3

Get your config

Project Overview ⚙ → "Project settings" → "Your apps" → Click the </> (Web) icon → Register app → Copy the firebaseConfig object

4

Paste your config here

🔒 Your config is stored only in this browser. Data in Firebase is shared with everyone who has the app URL.
